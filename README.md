# ❄️ PET-CARE-VETERINARY

A modern winter pet care service web application built with React, Firebase Authentication, Tailwind CSS, and DaisyUI. Users can explore winter pet services, view details, book services, and manage their profiles securely.

---

# 🌐 Live Website

https://pet-care-auth-services.web.app
---

# 🎯 Project Purpose

This project helps pet owners find reliable winter pet care services such as grooming, winter clothing, paw treatment, and veterinary support during cold weather.

Users can:

- Browse winter care services
- View service details
- Login/Register securely
- Book pet care services
- Update profile information
- Reset forgotten passwords

---

# 🚀 Key Features

- 🔐 Firebase Authentication (Email/Password + Google Login)
- 🛡️ Protected Routes
- 👤 User Profile & Update Profile
- 🎠 Winter-themed Hero Slider using Swiper
- 📦 Dynamic service cards from JSON data
- 🔥 Toast notifications
- 📱 Fully Responsive Design
- ❄️ Winter Pet Care Tips section
- 👨‍⚕️ Expert Veterinarians section
